# My dotfiles

My [dotfiles](https://dotfiles.github.io/) repository.
Feel free to use it as an inspiration.

# Installing

  1. Clone to `~/.dotfiles`
  2. Update submodules: `cd ~/.dotfiles && git submodule update --init --recursive`
  3. Run `~/.dotfiles/install.sh`
  4. Reload shell: `. ~/.zshrc`
